About

innfact-imagesharing is a simple Image sharing social networking website built in Ruby on Rails 4 and bootstrap 3.

Getting Started

1 Download as zip or tar.gz 2 Extract the project folder. 3 Cd into the project directory 4 Run bundle install to install required gems and dependencies. 5 Launch the server rails s.

That's it!! Edit and modify it according to your needs.
